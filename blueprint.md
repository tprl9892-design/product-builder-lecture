
# AI 기반 로또 번호 추첨 애플리케이션

## 개요

이 애플리케이션은 AI를 활용하여 사용자에게 로또 번호를 추천해주는 웹 애플리케이션입니다. 최신 웹 기술과 디자인 트렌드를 반영하여 사용자에게 매력적이고 직관적인 경험을 제공하는 것을 목표로 합니다.

## 스타일, 디자인 및 기능

### **디자인 철학**

- **모던하고 대담한 디자인**: 생생한 색상, 깊이감을 주는 그림자 효과, 그리고 세련된 타이포그래피를 사용하여 사용자에게 강렬한 첫인상을 남깁니다.
- **직관적인 사용자 인터페이스**: 사용자가 쉽게 상호작용할 수 있도록 버튼, 아이콘, 그리고 시각적 요소를 명확하고 일관성 있게 디자인합니다.
- **모바일 우선 반응형 디자인**: 모든 디바이스에서 최적의 경험을 제공하기 위해 반응형 웹 디자인 원칙을 따릅니다.

### **구현된 기능**

- **AI 로또 번호 생성**: 버튼 클릭 한 번으로 AI가 분석한 6개의 로또 번호와 보너스 번호 1개를 생성합니다.
- **애니메이션 효과**: 번호가 생성될 때 시각적인 즐거움을 더하기 위해 각 번호에 애니메이션 효과를 적용합니다.
- **시각적 피드백**: 생성된 각 번호는 고유한 색상의 공으로 표시되어 가시성을 높입니다.

### **기술 스택**

- **HTML5**: 웹 페이지의 구조를 정의합니다.
- **CSS3**: 모던 CSS 기능을 활용하여 스타일과 레이아웃을 구현합니다. (Flexbox, Grid, Custom Properties, Transitions)
- **JavaScript (ES6+)**: 동적인 기능과 사용자 상호작용을 처리합니다.

## 현재 변경 계획

1. **`index.html`**: 페이지 제목을 "AI 로또번호 추첨기"로 변경하고, 로또 번호를 표시할 컨테이너, 제목, 번호 디스플레이, 그리고 번호 생성 버튼을 포함한 기본 HTML 구조를 설정합니다.
2. **`style.css`**: 애플리케이션의 전체적인 디자인을 정의합니다. 배경, 컨테이너, 버튼, 그리고 번호 공에 대한 스타일을 포함하여 대담하고 현대적인 미학을 적용합니다.
3. **`main.js`**: 로또 번호 생성 로직을 구현합니다. 버튼 클릭 이벤트를 감지하고, 1부터 45까지의 숫자 중에서 7개의 고유한 번호를 무작위로 선택하여 화면에 애니메이션과 함께 표시하는 기능을 개발합니다.
