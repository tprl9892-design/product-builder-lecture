# AI 로또 번호 추첨 사이트

## 개요

AI를 이용하여 로또 번호를 추천해주는 웹 애플리케이션입니다. 사용자에게 행운의 번호를 제공하고, 최신 당첨 번호 정보를 보여줍니다.

## 디자인 및 기능

*   **레이아웃**: 중앙 정렬된 카드 형태의 레이아웃을 사용하여 사용자가 번호 추천에 집중할 수 있도록 합니다.
*   **스타일**: 파란색과 녹색을 중심으로 한 깔끔하고 현대적인 디자인을 적용했습니다. 추천 번호는 원형 배경으로 강조하여 시각적 효과를 높였습니다.
*   **기능**:
    *   '번호 생성' 버튼을 클릭하면 1부터 45까지의 숫자 중 6개의 고유한 번호를 무작위로 생성합니다.
    *   생성된 번호는 오름차순으로 정렬하여 표시됩니다.
    *   최신 당첨 번호 정보가 표시됩니다. (현재는 예시 데이터 사용)

## 현재 구현 계획

*   `index.html`: 웹 페이지의 기본 구조를 설정합니다.
*   `style.css`: 페이지의 디자인과 스타일을 정의합니다.
*   `main.js`: 로또 번호 생성 및 표시 로직을 구현합니다.
